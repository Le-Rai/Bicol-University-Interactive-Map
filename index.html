About this app

This website is not affiliated with or officially part of Bicol University. This is a personal project by Rainiel James Lazo, a BS Meteorology student fond of Mapping (QGIS). This map is still in progress and requires a lot of data that I am currently lacking, you may help me by clicking the "Help make this map better".

Goal

The primary goal of this project is to support incoming freshmen, students, faculty, and visitors with an accessible interactive platform that is able to locate their current location (w/consent) as well as search building names for seamless navigation across university grounds.
